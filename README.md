# 微信文件管理器

一个专为解决微信文件存储零散、深层、难以查找问题而设计的桌面应用工具。

## 功能特性

- 📁 **智能文件扫描**：自动扫描微信默认存储路径和自定义路径
- 📂 **分类管理**：按图片、视频、音频、文档、压缩文件等类型分类
- 🔍 **快速搜索**：支持按文件名搜索文件
- 🖼️ **图片缩略图**：直接预览图片文件
- 📄 **文档图标**：为不同类型的文档显示不同的图标
- 📦 **压缩文件支持**：专门的压缩文件分类和图标
- 🔧 **自定义设置**：支持自定义微信存储路径
- 📋 **批量管理**：支持多选、框选和批量操作
- 🎯 **智能过滤**：自动过滤系统文件和临时文件
- 📱 **响应式设计**：适配不同屏幕尺寸

## 安装方法

### 方法一：直接运行

1. 确保已安装 [Node.js](https://nodejs.org/zh-cn/)
2. 克隆或下载本项目
3. 在项目目录中运行：
   ```bash
   npm install
   npm start
   ```

### 方法二：使用打包版本

1. 在项目目录中运行：
   ```bash
   npm run build
   ```
2. 打包完成后，在 `dist` 目录中找到可执行文件
3. 双击运行即可

## 使用说明

1. **首次运行**：应用会自动扫描微信默认存储路径
2. **自定义路径**：如果微信安装在非默认位置，可以在设置中手动指定路径
3. **文件浏览**：在左侧导航栏选择分类查看对应类型的文件
4. **文件操作**：
   - 单击选择文件
   - 双击打开文件
   - 右键菜单查看更多操作
   - Ctrl键多选文件
   - Shift键选择连续文件
   - 拖动鼠标框选文件
5. **排序**：在右上角选择排序方式（按名称、大小、修改时间）

## 技术实现

- **框架**：Electron
- **语言**：JavaScript
- **文件系统**：Node.js fs模块
- **界面**：HTML5 + CSS3 + JavaScript
- **打包工具**：electron-builder

## 跨平台兼容性

- **Windows**：完全支持（Windows 7/10/11）
- **macOS**：支持，但需要手动设置微信路径
- **Linux**：理论支持，需要手动设置微信路径

## 常见问题

### 找不到微信文件
- 检查微信是否已登录并使用过
- 在设置中手动指定微信存储路径
- 确保有权限访问微信文件目录

### 扫描速度慢
- 微信文件较多时，首次扫描可能需要较长时间
- 后续扫描会使用缓存，速度会更快

### 无法打开文件
- 确保文件类型有对应的应用程序打开
- 检查文件是否损坏
